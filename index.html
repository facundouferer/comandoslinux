start linux

Recién llegade a LINUX? Entonces debería pegarle una mirada a estos comandos esenciales. 

COMANDOS AUXILIARES

sudo

argumento comando

Ejecuta un comando como super usuarios

man

argumento comando

Imprime el manual del comando especificado si es que éste existe.

usuarios

adduser

Añade un usuario.

deluser

Elimina un usuario.

passwd

Cambia la contraseña de un usuario

whoami

Imprime el nombre del usuario actual

who

Muestra el listado de usuarios con sesión iniciada. 

ARCHIVOS Y DIRECTORIOS

ls

Argumento opción directorio

lista el directorio indicado o el directorio actual si no se indica. 

mkdir

Argumento ruta

Crea un directorio en la ruta especificada y si sólo se le indica un nombre, lo crea en la ruta actual. 

cd

Argumento ruta

Cambia de directorio a la ruta especificada. 

pwd

Imprime la ruta actual. 

cp

Argumento origen destino

Copia un fichero a otra ruta. Para poder copiar un directorio completo es necesario hacerlo de forma recursiva con el argumento -r.

mv

Argumento origen destino

Mueve un fichero a otra ruta.

rm

Argumento fichero o directorio 

Elimina el fichero o directorio indicado Para borrar un directorio un directorio que no se encuentra vacío es necesario hacerlo de forma recursiva con -r.

cat

Argumento fichero

Imprime por pantalla el contenido del fichero indicado. 

find

Argumento ruta opción palabra_clave

Busca en la ruta indicada todos los ficheros que contengan la palabra clave. 

wheresi

Argumento fichero

Muy similar a find pero asumiendo que conoces el nombre del fichero actual. 

ln 

Argumento origen destino

Crea un enlace duro o sinbólico del fichero o directorio.

chmod

Argumento opcion destino

Cambia los permisos del directorio o fichero indicado. 

chown

Argumento usuario destino

Cambia el propietario del directorio o fichero indicado. 

PROCESOS

top

Muestra en tiempo real los procesos que se están ejecutando actualmente. 

ps

argumento opción

Muestra los procesos en ejecución. 

pidof

argumento proceso

Muestra el PID del proceso indicado

kill

argumento PID

Mata un proceso por PID.

killall

argumento nombre

Mata un proceso y todos sus hijos por nombre. 

DISCOS

mount

argumento unidad destino

Monta la unidad indicada. 

unmount

argumento unidad destino

Desmonta la unidad indicada. 

filtros

Los filtros se concatenan a otros comandos, generalmente. 

more

argumento opción

Filtra el contenido mostrado por pantalla hacia adelente. 

less

todo lo contrario de more. 

sort

ordena el contenido alfabeticamente. 

grep

Filtra el argumento siguiendo un patrón.

REDIRECCIONAMIENTOS OPERADORES Y PIPES

Símbolos reservados para concatenar comandos. 

>

Redirecciona la salida del comando hacia otro comando o fichero.

>>

Igual que > pero si se manda a un fichero no lo sobre escribe, agrega el contenido al final. 

<

Redirecciona un fichero como entrada a un comando. 

&&

AND. Si el primer comando finaliza con éxito, ejecuta el siguiente
